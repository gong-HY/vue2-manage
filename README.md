# 通用后台管理

## 项目安装
```
npm install
```

### 编译并启动开发服务器
```
npm run serve
```

### 编译并压缩代码用于生产环境
```
npm run build
```

### 检查并修复代码
```
npm run lint
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).
